# AI usage

Peerlane was built with AI-assisted development tools during the hackathon.

AI tools were used for:

- Drafting and iterating on the initial product concept.
- Generating and refining TypeScript, React, Docker, and documentation drafts.
- Reviewing implementation risks and suggesting fixes.
- Preparing demo copy and submission text.

Human direction was used for:

- Choosing the project scope and target prize.
- Evaluating design direction and rejecting overly generic UI patterns.
- Deciding the AXL topology and demo narrative.
- Running local checks and deciding what to include in the final submission.

The submitted repository contains the final implementation, run instructions,
architecture notes, and demo script needed to evaluate the project.
